Tree栏目树标签

适用版本

Previous_Releases_4.1.3 +

名称

栏目树标签

功能

该标签用于在页面中展示栏目树标签

语法

{dreamer-cms:tree root="A54547W2"}{/dreamer-cms:tree}

文件

cc.iteachyou.cms.taglib.tags.TreeTag

参数

参数名 描述
root 根节点栏目的编码（-1为顶级栏目）

底层字段

无

范例

{dreamer-cms:tree root="A54547W2" /}

效果

会返回栏目树html片段，样式可自行控制，如
